2주차 정리 내용

1. 배치방식

display / inline / inline-block

display

width 적용 가능. 줄바꿈 적용.

inline

width, margin 무시, 줄바꿈 무시.

inline-block

width, margin 적용 가능, 줄바꿈 무시.

2. 크기 조절

width / height / % px

width

가로 값 조정.

height

세로 값 조정.

%

부모 요소 기준에서 배율.

부모에 명시적으로 값이 없으면 적용 불가.

px

상수. 현재 값을 변경.

3. 정렬

text-align / margin: auto

  • text-align: 텍스트를 정렬.
  • margin: auto: 블록 요소 자체를 정렬.
  • justify-content / align-items: 블록 요소의 자식들을 정렬.
    • justify-content: 가로 정렬.
    • align-items: 세로 정렬.

4. 배경(background)

background-color / background-image / background-size ...

  • background-color / background-image: border 없이도 구분 시키기 가능.
  • background-position: background의 기준점 설정.
  • background-size: 이미지 크기 결정.
    • auto
    • cover
    • contain
    • px, % auto
