# Weather App
검색한 도시의 지금 날씨를 알 수 있는 간단한 Weather App. </br>
도시를 검색하면 날씨 카드가 전면에 띄워지는 형식. </br>
HTML, css, JaveScript 이용. </br>
</br>
## 주요 기능
```
- 날씨 검색란에서 검색하면 날씨 카드 제공.
  날씨 카드 내부에 도시명, 날씨 이미지, 온도, 날씨 상태(맑음, 비, 흐림 등) 표기
- 검색 시, 최근에 검색했던 기록이 날씨 카드 상단에 표기.
  해당 최근 검색 기록을 클릭하면 날씨를 보여주며, 그 옆의 x 표기를 누르면 최근 검색 기록 제거.
- 날씨 검색 시, 배경 이미지가 날씨에 맞게 변경.
- localStorage를 이용한 데이터 유지.
```
